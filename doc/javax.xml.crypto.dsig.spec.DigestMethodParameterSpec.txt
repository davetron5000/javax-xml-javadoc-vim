*javax.xml.crypto.dsig.spec.DigestMethodParameterSpec_Description*

A specification of algorithm parameters for a (|javax.xml.crypto.dsig.DigestMethod|) algorithm. The purpose of this interface 
is to group (and provide type safety for) all digest method parameter specifications. All digest method parameter 
specifications must implement this interface.
