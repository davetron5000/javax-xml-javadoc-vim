*javax.xml.crypto.dsig.spec.TransformParameterSpec_Description*

A specification of algorithm parameters for a (|javax.xml.crypto.dsig.Transform|) algorithm. The purpose of this interface is 
to group (and provide type safety for) all transform parameter specifications. All transform parameter specifications must 
implement this interface.
