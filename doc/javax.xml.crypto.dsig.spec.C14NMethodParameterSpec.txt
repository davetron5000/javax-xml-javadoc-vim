*javax.xml.crypto.dsig.spec.C14NMethodParameterSpec_Description*

A specification of algorithm parameters for a (|javax.xml.crypto.dsig.CanonicalizationMethod|) Algorithm. The purpose of this 
interface is to group (and provide type safety for) all canonicalization (C14N) parameter specifications. All canonicalization 
parameter specifications must implement this interface.
