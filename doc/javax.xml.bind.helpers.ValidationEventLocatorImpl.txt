*javax.xml.bind.helpers.ValidationEventLocatorImpl_Description*

Default implementation of the ValidationEventLocator interface. 

JAXB providers are allowed to use whatever class that implements the ValidationEventLocator interface. This class is just 
provided for a convenience.
