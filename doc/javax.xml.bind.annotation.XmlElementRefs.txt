*javax.xml.bind.annotation.XmlElementRefs_Description*

Marks a property that refers to classes with (|javax.xml.bind.annotation.XmlElement|) or JAXBElement. 

Compared to an element property (property with (|javax.xml.bind.annotation.XmlElement|) annotation), a reference property has a 
different substitution semantics. When a sub-class is assigned to a property, an element property produces the same tag name 
with @xsi:type, whereas a reference property produces a different tag name (the tag name that's on the the sub-class.) 

This annotation can be used with the following annotations: (|javax.xml.bind.annotation.adapters.XmlJavaTypeAdapter|) , 
(|javax.xml.bind.annotation.XmlElementWrapper|) .
